- Plugin ID: `todo`
- Version: `1.5.6`
- Source: `docs/plugin-packages/todo`
- App compatibility: `>=1.5.6`

Install URL:

```text
https://github.com/GigaPuddings/snippets-code-plugin-todo/archive/refs/heads/main.zip
```

Versioned release URL:


This repository is synchronized from the main application repository by

package, then synchronized again.
